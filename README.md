## About
**Queens** is a logic game available on the professional networking site LinkedIn. The objective of the game is to place queens on a colored square board so that there is only one queen in each row, column, and color region. Additionally, a queen cannot be placed adjacent to another queen, including diagonally. This repository features a solver for the Queens logic game, implemented using a brute-force search algorithm.
